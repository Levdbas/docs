### thumbnail
`thumbnail( )`

**returns:** `string` the public thumbnail url

Gets the post thumbnail image object

**Twig**
```twig
{% for item in menu.items %}
    <li><a href="{{ item.link }}"><img src="{{ item.thumbnail }}"/></a></li>
{% endfor %}
